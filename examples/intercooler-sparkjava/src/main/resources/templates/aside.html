intercooler.js lets you use familiar, declarative HTML attributes to implement advanced, dynamic interfaces in web applications with a minimum of complexity. intercooler.js lets you extend the HTML vocabulary for your application without requiring much, if any, explicit javascript. The resulting environment is extraordinarily simple, expressive, readable, and maintainable.
